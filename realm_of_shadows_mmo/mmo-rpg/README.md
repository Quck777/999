# 🎮 Realm of Shadows — Браузерная MMORPG

Текстовая многопользовательская RPG с глубоким фэнтезийным сеттингом.

## 🚀 Быстрый старт

### Демо-режим (без сервера)
Просто откройте `public/index.html` в браузере — игра автоматически запустится в демо-режиме со всеми функциями!

### Через PHP встроенный сервер
```bash
cd public
php -S localhost:8080
```
Откройте http://localhost:8080 в браузере.

### Полноценная установка с базой данных
1. Скопируйте `.env.example` в `.env` и настройте подключение к БД
2. Создайте базу данных MySQL
3. Запустите миграции (в разработке)
4. Настройте веб-сервер (nginx/Apache) на папку `public/`

## 🎯 Особенности

### Игровой мир
- **14 уникальных монстров** от волков до титанов бури
- **10 локаций** для исследования
- **18 предметов** разной редкости
- **20 достижений** для выполнения
- **18 рецептов крафта**

### Боевая система
- Пошаговые бои с расчётом урона
- Критические удары и уклонения
- Система способностей
- Лут и награды

### Социальные функции
- Глобальный, локальный, торговый и гильдийный чаты
- Боты для демо-режима
- Система друзей и гильдий

### Экономика
- Золото и торговля
- Аукцион игроков
- Магазин NPC
- Крафт предметов

## 🛠 Технологии

## 🎮 Как играть

1. **Регистрация/Вход** — создайте персонажа или войдите в демо-режиме
2. **Выбор локации** — путешествуйте между городами и опасными зонами
3. **Бой** — сражайтесь с монстрами, получайте опыт и лут
4. **Инвентарь** — экипируйте предметы, используйте зелья
5. **Крафт** — создавайте новые предметы из ресурсов
6. **Чат** — общайтесь с другими игроками (или ботами в демо)
7. **Достижения** — выполняйте задания для наград

## 🐛 Отладка

Игра автоматически определяет работу без базы данных и переключается в демо-режим. Все API запросы возвращают mock-данные.

Для просмотра логов:
```bash
tail -f logs/app.log
```
